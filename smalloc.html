SMALLOC - SIMPLE MALLOC


One of the operation system course (234123) projects was to write a simple malloc function.
Our smalloc was implemented using sbrk and mmap function according to the size that requested.
the projects contain 4 main functon:

  • "smalloc size": searches for a free block which size bigger than "size" bytes and split it if there are more than 128B extra bytes to fit the "size" or allocates if none are found, if size smaller than 128KB use sbrk, otherwise, use mmap.
  • "scalloc num size": searches for a block of "num" elements, each "size" bytes then set all to 0 or allocates if none are found.
  • "sfree pointer": releases the usage of the block that starts with the pointer, if the prev or next block is free then merge them.
  • "srealloc old_pointer size": if "size" is smaller than the current blockâ€™s size then reuse the same block. Otherwise split the first block with a size bigger than the requested, and finally if nothing worked\ allocate new memory. copy the content of old space into new space and frees the old space.


We implement our smalloc using c++ programming language, we create an object for memory allocation which contains the allocated size, if it free, a pointer to the start of the block.
We chose to save the memory allocated in the linked list like the real one.
